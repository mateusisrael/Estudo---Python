Curso de Python - Aula 06 - Modos primitivos
=========Tipos Primitivos=========
int = Valores verdadeiros (inteiros
float = valores decimais
bool = True or False (sempre que usar o True, use com 'T' maiúsculo, o mesmo para o False)
str = Entre áspas(mesagens) - Exempo: 'Olá' ou/e '7.5'.


===Sístaxe do print no Python 3:

n1 = input('Escreva um número ')
n2 = input('Escreva mais um número ')
s = n1 + n2
print('A soma vale {}'.format(s))
